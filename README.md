# ARCHE Big Screen

Static big-screen presentation pages for the ARCHE chemical reaction mechanism discovery showcase.

## Pages

- `index.html` - ARCHE cover and architecture detail page
- `chipdesign-agent.html` - preserved ChipDesign Agent cover page

## Run Locally

Open `index.html` directly in a browser, or serve the folder with any static file server.

```bash
python3 -m http.server 8080
```
